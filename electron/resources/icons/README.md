# App Icons

Place your application icons here:

- **icon.ico**: Windows icon (256x256)
- **icon.png**: Linux icon (512x512)

For production builds, replace these placeholder files with actual icons.

## Generating Icons

You can use tools like:
- https://icon.kitchen/ (web-based)
- https://www.electron.build/icons (electron-builder guide)
- ImageMagick for command-line conversion

## Requirements

- Windows (.ico): 256x256 pixels or larger
- Linux (.png): 512x512 pixels recommended
- macOS (.icns): 1024x1024 pixels for best quality
